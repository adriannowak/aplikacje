##  Import/Export danych do bazy MySQL


Zapisywanie kopii bazy danych:

 - `mysqldump.exe -u root -p -x -r DANE.SQL BAZA`
 
Przywracanie kopii:
 - `CREATE DATABASE BAZA`
 - `mysql -u root -p BAZA < DANE.SQL`
